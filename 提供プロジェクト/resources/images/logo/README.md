# Logo

Certify LMS のロゴアセット。

| ファイル | 用途 |
|---|---|
| `logo-mark.svg` | シンボル単体 (64×64) — favicon 以外の全用途 |
| `logo-wordmark.svg` | シンボル + "Certify LMS" 文字 (280×64) — サイドバー TopBar / マーケティング |
| `favicon.svg` | 32×32 — ブラウザタブ |

**コンセプト**:
- 角丸の正方形に **Sunrise Amber → Mint → Teal Aqua** のグラデーション (南国の朝の海)
- 中央の白い "C" 開口 — Certify の頭文字 + 「開く」感
- 中央に小さな白点 — 「開いて受け取る」象徴
- 左上の薄い楕円 — 冷えた缶 / 海面のハイライト

**使い方**:
- ロゴ色は **変更しない**。グレースケール版は許可しない (本ブランドのコアは色そのもの)
- 黒背景に置く場合: ロゴ周りに `padding: 8px` を必ず取る (Amber 側が沈むため)
- 最小サイズ: シンボル 24px / wordmark 120px

**書き出し**: SVG マスタ。PNG が必要な場合は ImageMagick / svgo で 2x/3x を生成。
