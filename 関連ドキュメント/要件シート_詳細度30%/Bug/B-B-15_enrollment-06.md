# B-B-15 コーチの受講生一覧に担当外の受講生まで表示される

## メタ情報

| 項目 | 値 |
|---|---|
| チケット ID | `B-B-15` |
| Feature 連番 | `enrollment-06` |
| Feature | enrollment |
| 種別 | Bug |
| サブカテゴリ | 認可・認証(担当スコープ漏れ) |
| 難易度 | Basic |
| 工数 (h) | 3 |
| 依存チケット | (なし) |

## 概要

コーチ(coach)が受講登録の管理一覧を開くと、本来は自分が担当として割り当てられた資格の受講生のみが表示されるべきところ、担当していない資格の受講生まで一覧に表示される。担当外の受講生の氏名・所属資格・進捗などをコーチが閲覧できてしまう、認可(表示範囲の絞り込み)の不備である。

## 再現手順

**前提**: コーチでログイン済み。そのコーチが担当する資格と、担当しない資格の双方に受講生が登録されている。

1. コーチで受講登録の管理一覧を開く
2. → 自分が担当する資格の受講生だけでなく、担当していない資格の受講生も一覧に表示される(資格の列を見ると担当外の資格が混在している)

## 期待する結果

コーチの受講登録管理一覧には、そのコーチが担当として割り当てられた資格に属する受講生のみが表示される。担当外の資格の受講生は一覧に出ない。

## 実際の結果

担当資格による絞り込みが効かず、全資格の受講生が一覧に表示される。コーチが担当外の受講生の情報まで閲覧できる状態になっている。
