# T-B-02 コーチダッシュボードの担当受講生一覧のクエリ最適化

## 概要

コーチ(coach)のダッシュボードに表示される担当受講生一覧は、担当受講生が増えるほど表示が重くなる。担当受講生が増えても表示が重くならないように一覧の表示を最適化する。一覧の各行に出る関連情報(受講生氏名 / 担当資格名 / 最終活動日時)や並びは、改修の前後で変えない。

## 要件

- 担当受講生が増えても、コーチダッシュボードの担当受講生一覧の表示が重くならないように最適化する
- 一覧の各行に出る関連情報(受講生氏名 / 担当資格名 / 最終活動日時)・並び順・絞り込み仕様は、改修の前後で変えない

## スコープ外

- 管理者のユーザー管理画面の同様の表示最適化 — 本チケットでは扱わない(別タスク)
- ダッシュボードの他のカード(プラン情報 / 通知 / 学習進捗 等)の最適化
- キャッシュ層の導入
- 表示内容・並び順・絞り込みを変える変更(振る舞いは不変)
