# B-B-16 修了したユーザーがプラン機能にアクセスできてしまう

## メタ情報

| 項目 | 値 |
|---|---|
| チケット ID | `B-B-16` |
| Feature 連番 | `auth-05` |
| Feature | auth(学習有効性ガード ミドルウェア) |
| 種別 | Bug |
| サブカテゴリ | 認可・認証(Middleware チェック漏れ) |
| 難易度 | Basic |
| 工数 (h) | 3 |
| 依存チケット | (なし) |

## 概要

修了(graduated)したユーザーが、本来は受講中(在籍中)のユーザーだけが使えるプラン機能(教材閲覧 / 問題演習 / 模試 / 面談 / チャット / 質問掲示板 / AI 相談)にアクセスできてしまう。プラン期間が満了しても各機能が利用でき、利用制限が効いていない。

## 再現手順

**前提**: 修了(graduated)状態のユーザーでログイン済み。

1. 修了ユーザーのまま、プラン機能(例: 教材閲覧 / 模試 / 面談予約 / チャット / 質問掲示板 / AI 相談)の画面にアクセスする
2. → 403 で弾かれず、プラン機能が利用できてしまう
3. (対比)受講中のユーザーでは同じ機能に正常にアクセスできる(こちらは正しい)

## 期待する結果

プラン機能は受講中(在籍中)のユーザーのみ利用でき、修了ユーザーがアクセスすると 403 が返る。一方で、プロフィール / 修了証ダウンロード / 通知 / ダッシュボード / 受講登録の閲覧などは修了後も引き続き利用できる(これらはプラン機能ガードの対象外)。

## 実際の結果

修了ユーザーがプラン機能の各画面にアクセスでき、プラン期間満了後も教材閲覧・模試・面談・チャット・質問掲示板・AI 相談などが利用できてしまう。

## 受け入れ条件

- [ ] 修了ユーザーがプラン機能(教材閲覧 / 問題演習 / 模試 / 面談 / チャット / 質問掲示板 / AI 相談)にアクセスすると 403 が返り、受講中のユーザーは従来どおりアクセスできる(正常系が維持されている)
- [ ] 修了ユーザーでも、プラン機能ガードの対象外(プロフィール / 修了証ダウンロード / 通知 / ダッシュボード / 受講登録の閲覧)には引き続きアクセスできる
- [ ] 本チケットの機能に対するテスト (Unit / Feature 等) が実装されている

## 実装方針(参考)

### 原因

- **主要ファイル**: `app/Http/Middleware/EnsureActiveLearning.php`(学習有効性ガード、`active-learning` エイリアスでプラン機能の各ルート群に適用)。
- **仕込み内容**: 「利用状態が受講中(`UserStatus::InProgress`)でなければ 403」の判定が抜けている(または「ユーザーが存在するか」だけの判定に縮小されている)。このミドルウェアは教材閲覧 / 問題演習 / 模試 / 面談予約 / 追加面談購入 / チャット / 質問掲示板 / AI 相談のルート群に適用され、受講中以外(修了 / 退会 / 招待中)を一括で拒否する単一判定。判定が抜けると修了 `Graduated` 等でもプラン機能を通過してしまう。プロフィール / 修了証 DL / 通知 / ダッシュボード / 受講登録の閲覧ルートにはこのミドルウェアが付かないため影響しない。
- **修正範囲**: ガードの判定を「利用状態が受講中(`InProgress`)でなければ 403」に戻す。Middleware 内 1 行(状態の不等式判定)の修正で完結。

## 補足

### 想定 Q&A

| 質問 | 回答 |
|---|---|
| プラン機能にアクセスできる利用状態は? | 受講中(在籍中)のみ。修了 / 招待中 / 退会はプラン機能を利用できない(403) |
| 修了ユーザーが引き続き使える機能は? | プロフィール / 修了証ダウンロード / 通知 / ダッシュボード / 受講登録の閲覧。これらは学習有効性ガードの対象外で、修了後もアクセスできる |
| 弾かれたときの HTTP ステータスは? | 403。プラン期間が満了している旨のメッセージを表示する |
| 退会ユーザーも同じガードで弾かれるか? | 弾かれる。ガードは「受講中以外」を一括拒否するため、修了・退会・招待中はいずれも 403 になる |
| ログインそのものは修了ユーザーでも可能か? | 可能。ログインは修了ユーザーも通る(プロフィール / 修了証 DL のため)。プラン機能だけがこのガードで制限される |
| 質問掲示板(qa-board)やチャットも対象か? | 対象。これらも受講中のみ利用できるプラン機能で、修了ユーザーは 403。教材・模試・面談などと同じ扱い |
