## 概要

修了(graduated)したユーザーが、本来は受講中(在籍中)のユーザーだけが使えるプラン機能(教材閲覧 / 問題演習 / 模試 / 面談 / チャット / 質問掲示板 / AI 相談)にアクセスできてしまう。プラン期間が満了しても各機能が利用でき、利用制限が効いていない。

## 再現手順

**前提**: 修了(graduated)状態のユーザーでログイン済み。

1. 修了ユーザーのまま、プラン機能(例: 教材閲覧 / 模試 / 面談予約 / チャット / 質問掲示板 / AI 相談)の画面にアクセスする
2. → 403 で弾かれず、プラン機能が利用できてしまう
3. (対比)受講中のユーザーでは同じ機能に正常にアクセスできる(こちらは正しい)

## 期待する動作

プラン機能は受講中(在籍中)のユーザーのみ利用でき、修了ユーザーがアクセスすると 403 が返る。一方で、プロフィール / 修了証ダウンロード / 通知 / ダッシュボードなどは修了後も引き続き利用できる(これらはプラン機能ガードの対象外)。

## 実際の動作

修了ユーザーがプラン機能の各画面にアクセスでき、プラン期間満了後も教材閲覧・模試・面談・チャット・質問掲示板・AI 相談などが利用できてしまう。

## 受け入れ条件

- [ ] 修了ユーザーがプラン機能(教材閲覧 / 問題演習 / 模試 / 面談 / チャット / 質問掲示板 / AI 相談)にアクセスすると 403 が返る
- [ ] 受講中のユーザーはプラン機能に従来どおりアクセスできる(正常系が維持されている)
- [ ] 修了ユーザーでも、プラン機能ガードの対象外(プロフィール / 修了証ダウンロード / 通知 / ダッシュボード)には引き続きアクセスできる

## 実装方針

> **参考設計の一例**。受け入れ条件を満たせれば実装手段は問わない。受講生は提供 PJ コード + ヒアリングで自分の調査・修正方針を組み立てる。

### 主要 URL

| メソッド | パス | 振る舞い |
|---|---|---|
| GET | `/learning/...` / `/quiz/...` / 模試 / `/meetings/...` / `/chat-rooms/...` / `/qa-board/...` / `/ai-chat/...` | プラン機能(受講中のみ。修了 / 退会は 403 であるべき) |
| GET | `/dashboard` / `/settings/profile` / `/certificates/{certificate}/download` / `/notifications` | プラン機能ガード対象外(修了後もアクセス可、対比用) |

### 原因箇所メモ

- 原因の主要ファイル: `app/Http/Middleware/EnsureActiveLearning.php`(学習有効性ガード)
- 関連: このミドルウェアはプラン機能の各ルート群に適用され、「利用状態が受講中(`in_progress`)でなければ 403」で弾く。受講中以外(修了 / 招待中 / 退会)を一括で拒否する単一判定。プロフィール / 修了証 DL / 通知 / ダッシュボードのルートにはこのミドルウェアが付かない設計
- 仕込み内容(Step 4 引き算): ミドルウェアの利用状態判定行を削除する(または「ユーザーが居るか」だけの判定に縮小する)。これにより修了 / 退会など受講中以外の状態でもプラン機能を通過してしまう
- 受講生が辿るべき修正範囲: ガードの判定を「利用状態が受講中でなければ 403」に戻す。修了ユーザーでプラン機能に入れてしまうがプロフィール等には影響しない、という切り分けから、プラン機能ルート群に適用される学習有効性ガードの判定漏れを発見する流れ

## 補足

### 想定ヒアリング Q&A

| 質問 | 回答 |
|---|---|
| プラン機能にアクセスできる利用状態は? | 受講中(在籍中)のみ。修了 / 招待中 / 退会はプラン機能を利用できない(403) |
| 修了ユーザーが引き続き使える機能は? | プロフィール / 修了証ダウンロード / 通知 / ダッシュボード。これらは学習有効性ガードの対象外で、修了後もアクセスできる |
| 弾かれたときの HTTP ステータスは? | 403。プラン期間が満了している旨のメッセージを表示する |
| 退会ユーザーも同じガードで弾かれるか? | 弾かれる。ガードは「受講中以外」を一括拒否するため、修了・退会・招待中はいずれも 403 になる |
| ログインそのものは修了ユーザーでも可能か? | 可能。ログインは修了ユーザーも通る(プロフィール / 修了証 DL のため)。プラン機能だけがこのガードで制限される |
