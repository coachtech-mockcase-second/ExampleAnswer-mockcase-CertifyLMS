## 概要

受講生(student)が、公開停止(アーカイブ)された資格の教材(Part / Chapter / Section)を引き続き閲覧できてしまう。本来は資格が公開停止になった時点で、その資格の教材閲覧画面は閲覧不可になるべきだが、受講登録済みの受講生は教材詳細を開けてしまう。

## 再現手順

**前提**: 受講生 A でログイン済み。受講生 A は資格 X に受講登録済み(受講中)。その後、管理者が資格 X を公開停止(アーカイブ)した。

1. 受講生 A で資格 X の教材詳細(Part / Chapter / Section いずれか)を開く
2. → 資格 X は公開停止済みにもかかわらず、教材詳細が 200 で表示される
3. 公開中の資格 Y の教材は従来どおり表示される(対比用)

## 期待する動作

受講生が公開停止(アーカイブ)された資格の教材詳細(Part / Chapter / Section)にアクセスすると 404 が返り、教材は閲覧できない。資格が公開中の場合のみ、受講登録済みの受講生は教材を閲覧できる。

## 実際の動作

資格が公開停止されていても、受講登録(受講中 / 合格)が残っていれば教材詳細が表示され、公開停止後も教材の閲覧が継続できてしまう。

## 受け入れ条件

- [ ] 受講登録済みの受講生が、公開停止(アーカイブ)された資格の教材詳細(Part / Chapter / Section)にアクセスすると 404 が返る
- [ ] 公開中の資格については、受講登録済みの受講生が従来どおり教材詳細を閲覧できる(本修正で正常系が壊れていないこと)

## 実装方針

> **参考設計の一例**。受け入れ条件を満たせれば実装手段は問わない。受講生は提供 PJ コード + ヒアリングで自分の調査・修正方針を組み立てる。

### 主要 URL

| メソッド | パス | 振る舞い |
|---|---|---|
| GET | `/learning/parts/{part}` | 教材閲覧: Chapter 一覧 |
| GET | `/learning/chapters/{chapter}` | 教材閲覧: Section 一覧 |
| GET | `/learning/sections/{section}` | 教材閲覧: Section 詳細(本文) |

### 原因箇所メモ

- 原因の主要ファイル: `app/UseCases/Learning/ShowPartAction.php` / `ShowChapterAction.php` / `ShowSectionAction.php`(受講生向け教材閲覧の各取得 Action)
- 関連: `CertificationStatus`(`draft` / `published` / `archived`)/ `Part` → `certification` リレーション。既存の各 Action には「Part / Chapter / Section が公開状態(`ContentStatus::Published`)でなければ 404」の判定が既にあり、本件はそこに「親資格が公開中(`CertificationStatus::Published`)でなければ 404」の判定を **加えている**(模範解答 PJ 側で正解像として整備済み)
- 仕込み内容(Step 4 引き算): 上記 3 Action から「親資格が公開中でなければ 404」の判定ブロックを削除する。これにより、教材階層自体が公開済みであれば、親資格が公開停止でも教材が閲覧できてしまう
- 受講生が辿るべき修正範囲: 教材閲覧の取得処理に「親資格が公開中でなければ 404」の絞り込み(ガード)を戻す。`BrowseController` の各 show メソッドから対応する Action を辿り、教材階層の公開判定はあるが親資格の公開判定が抜けていることを発見する流れ

## 補足

### 想定ヒアリング Q&A

| 質問 | 回答 |
|---|---|
| 公開停止資格の教材にアクセスしたときの HTTP ステータスは? | 404。教材階層が非公開のときと同じ扱い(存在を伏せる) |
| 受講登録済み(受講中 / 合格)なら公開停止後も閲覧できるべきでは? | できない。資格が公開停止された時点で、受講登録の有無に関わらず教材閲覧は不可。受講生の進捗記録自体は残るが、教材本文・教材一覧の閲覧導線は塞ぐ |
| 下書き状態の資格はどうなるか? | 公開中以外(下書き / アーカイブ)はすべて教材閲覧不可(404)。「公開中のときのみ閲覧可」という判定 |
| この Bug と「未登録資格の教材直叩き(B-B-09)」の違いは? | 本件は「受講登録済みだが資格が公開停止」のケース(絞り込み漏れ)。B-B-09 は「そもそも受講登録していない資格」のケース(認可判定の呼び出し漏れ)。発生レイヤーが異なる |
