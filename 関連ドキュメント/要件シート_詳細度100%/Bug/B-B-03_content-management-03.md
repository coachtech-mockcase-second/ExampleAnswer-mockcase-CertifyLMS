# B-B-03 公開停止した資格の教材が受講生に閲覧できてしまう

| Feature 連番 | `content-management-03` |
| Feature | content-management(受講生向け教材閲覧 / learning) |
| 種別 | Bug |
| サブカテゴリ | データ(絞り込み漏れ) |
| 難易度 | Basic |
| 工数 (h) | 3 |

> **※ Basic 範囲外への例外注記**: 既存実装では受講生向け教材閲覧の取得処理を `app/UseCases/Learning/Show{Part,Chapter,Section}Action.php` の Action に分離しているため、原因箇所は Action ファイル内にある。Basic 受講生は通常 Controller 範囲で実装してよく、その場合は対応する Controller メソッド(`BrowseController::showPart` / `showChapter` / `showSection`)を修正対象とする。いずれの構成でも受け入れ条件の振る舞いを満たせばよい。

## 概要

受講生(student)が、公開停止(アーカイブ)された資格の教材(Part / Chapter / Section)を引き続き閲覧できてしまう。本来は資格が公開停止になった時点で、その資格の教材閲覧画面は閲覧不可になるべきだが、受講登録済みの受講生は教材詳細を開けてしまう。

## 再現手順

**前提**: 受講生 A でログイン済み。受講生 A は資格 X に受講登録済み(受講中)。その後、管理者が資格 X を公開停止(アーカイブ)した。

1. 受講生 A で資格 X の教材詳細(Part / Chapter / Section いずれか)を開く
2. → 資格 X は公開停止済みにもかかわらず、教材詳細が表示される
3. 公開中の資格 Y の教材は従来どおり表示される(対比用)

## 期待する結果

受講生が公開停止(アーカイブ)された資格の教材詳細(Part / Chapter / Section)にアクセスすると 404 が返り、教材は閲覧できない。資格が公開中の場合のみ、受講登録済みの受講生は教材を閲覧できる。

## 実際の結果

資格が公開停止されていても、受講登録(受講中 / 合格)が残っていれば教材詳細が表示され、公開停止後も教材の閲覧が継続できてしまう。

## 受け入れ条件

- [ ] 受講登録済みの受講生が、公開停止(アーカイブ)された資格の教材詳細(Part / Chapter / Section)にアクセスすると 404 が返る
- [ ] 公開中の資格については、受講登録済みの受講生が従来どおり教材詳細を閲覧できる(本修正で正常系が壊れていない)
- [ ] 本チケットの機能に対するテスト (Unit / Feature 等) が実装されている

## 実装方針(参考)

### 原因

- **主要ファイル**: `app/UseCases/Learning/ShowPartAction.php` / `ShowChapterAction.php` / `ShowSectionAction.php`(受講生向け教材閲覧の各取得)。Basic で Controller 内完結とする場合は `BrowseController::showPart` / `showChapter` / `showSection`。
- **仕込み内容**: 各取得処理には「教材階層(Part / Chapter / Section)自体が公開状態(`ContentStatus::Published`)でなければ 404」の判定はあるが、「親資格が公開中(`CertificationStatus::Published`)でなければ 404」の判定が抜けている。このため教材階層が公開済みであれば、親資格が公開停止(アーカイブ / 下書き)でも受講登録済み受講生に教材が見えてしまう。
- **修正範囲**: 各取得処理に「親資格が公開中でなければ 404」のガードを追加する。教材階層の公開判定の直後に資格状態の判定を 1 ブロック足す形で完結。

## 補足

### 想定 Q&A

| 質問 | 回答 |
|---|---|
| 公開停止資格の教材にアクセスしたときの HTTP ステータスは? | 404。教材階層が非公開のときと同じ扱い(存在を伏せる) |
| 受講登録済み(受講中 / 合格)なら公開停止後も閲覧できるべきでは? | できない。資格が公開停止された時点で、受講登録の有無に関わらず教材閲覧は不可。受講生の進捗記録自体は残るが、教材本文・教材一覧の閲覧導線は塞ぐ |
| 下書き状態の資格はどうなるか? | 公開中以外(下書き / アーカイブ)はすべて教材閲覧不可(404)。「公開中のときのみ閲覧可」という判定 |
